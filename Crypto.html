Cryptocurrency Price Alert

This project alerts the user via email when the price of a Cryptocurrency (Bitcoin used in my code) fluctuates



Technical Details

  • Python libraries such as:
    1. bs4 (Beautiful Soup) to parse HTML
    2. requests to fetch price of cryptocurrency
    3. smtplib (Simple Mail Transfer Protocol Client) to send email
    4. ssl (Secure Sockets Layer) for secure connection to server
  • MIMEMultipart object was created to store details of reciever, sender and subject of email
  • HTML MIMEText object was created to store content of email
